# :hourglass: Tracky
Time and activity tracking prototype for the [Hackaburg 2019](https://hackaburg.de/)

## Modules
* `android` Tracker app that sends the currently active app to the webserver in `api-java`
* `win-daemon` Tracker Windows app that transmitts window focus changes to the webserver in `api-java`
* `api-java` Spring Boot Backend to provide a Database with a Rest endpoint
* ~~`api`~~ deprecated backend, built with node
* `frontend` Frontend with Vue.js, Quasar and node.js
